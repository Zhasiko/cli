# cli
cli task for "improvado"

Небольшой инструмент для получения данных о друзьях по идентификатору пользователя
  Социальная сеть Вконтакте.
                                                                            
  Для работы программы необходимо получить и ввести API ключ ВК. Ты можешь
  получить ключ API здесь, вам нужно зарегистрироваться и создать приложение. Затем
  посмотрите в настройках приложения поле Сервисный ключ доступа
  https://vk.com/editapp?act=create.
                                                                            
  Затем нужно ввести ID пользователя ВК, например: 232090689

  Затем формат файла: csv, json, tsv. По умолчанию csv

  Наконец, путь и имя файла, в котором вы хотите сохранить свои данные: C:/python/a, по умолчанию ./report
  
В общем должно получиться примерно так: python  cli.py --api-key  236a6002236a6002236a6002cc20793c452236a236a600247426d7f970a82d55fc5d819  232090689  json
